# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-01-14

### Added

- Initial release with 5 commands and 8 subagents

**Commands:**
- `/daily-brief` - Personalized morning news briefing
- `/daily-checkin` - End-of-day reflection and planning
- `/weekly-checkin` - Context-aware weekly metrics tracking
- `/brain-dump-analysis` - Extract insights from stream-of-consciousness
- `/newsletter-research` - Research competitors and draft newsletters

**Subagents:**
- `interest-analyzer` - Identifies interests from project files
- `news-curator` - Searches and filters recent news
- `daily-reflection` - Analyzes mood and productivity patterns
- `metrics-analyst` - Creates visual metric reports
- `insight-extractor` - Finds patterns in brain dumps
- `brain-dump-analyst` - Creates visual mind maps
- `content-researcher` - Analyzes competitor content
- `newsletter-writer` - Drafts newsletters in user's voice

**Documentation:**
- Comprehensive README with examples
- Contributing guidelines
- Installation script
